## 四、圆盘间的Mobius变换

### 1. Mobius变换的自由度与三点决定

Mobius变换分子分母同乘一非零复数$k$；

$$\frac{az+b}{cz+d}\to \frac{kaz+kb}{kcz+kd}$$

得到的是和原来相同的变换. 先前已经证明Mobius变换

$$\frac{az+b}{cz+d}$$

和商群

$$\mathbb{C^*}\cdot\left\{\left(\begin{matrix}
    a&b\\c&d
\end{matrix}\right):ad\ne bc\right\}$$

有同构关系，我们不妨从商群的每一个陪集中挑出一个代表元，并用其对应的Mobius变换统一表示和它相同的Mobius变换. 具体地，对于某个陪集

$$\mathbb{C^*}\left(\begin{matrix}
    a&b\\c&d
\end{matrix}\right)$$

选择其中的元素

$$k\left(\begin{matrix}
    a&b\\c&d
\end{matrix}\right)$$

使得此矩阵行列式为$1$. 从Mobius变换群的角度来看，这相当于额外增加了限制条件$ad-bc=1$. 四个参数与一个限制条件，表明Mobius变换有三个自由度. 

**定理（定理4.4.1）** 给定互不相同的$h,f,g\in \overline{\mathbb{C}}$和互不相同的$H,F,G\in \overline{\mathbb{C}}$，存在唯一Mobius变换$L$使得

$$L(h)=H,\quad L(f)=F,\quad L(g)=G.$$

**证明** 存在性：考虑

$$
L _ { 1 } : z \mapsto \frac { z - h } { z - f } ~ \frac { g - f } { g - h }
$$

$$
L _ { 2 } : w \mapsto \frac { w - H } { w - F } ~ \frac { G - T } { G - H }
$$

显然$h,f,g$中有一个为$\infty$时，$L_1$也是良定的；$H,F,G$中有一个为$\infty$时，$L_2$也是良定的. 

则

$$
L _ { 1 } \left( h \right) = L _ { 2 } \left( H \right) = 0 \,, L _ { 1 } \left( f \right) = L _ { 2 } \left( F \right) = \infty \,, L _ { 1 } \left( g \right) = L _ { 2 } \left( G \right) = 1
$$

令$L=L_2^{-1}\circ L_1$，由于Mobius变换的逆也是Mobius变换，Mobius变换群对复合封闭，则$L$也是Mobius变换，且符合要求. 

唯一性：设Mobius变换$\lambda$满足题目条件，令$\mu = L _ { 2 } \circ \lambda \circ { L _ { 1 } } ^ { - 1 }$，则$0,1,\infty$是$\mu$的不动点，设$\mu(z)=(az+b)/(cz+d)$，则代入得$b/d=0$，$a+b=c+d$，$a/c=\infty$，则$b=0$，$c=0$，$a=d$，则$\mu$是恒等变换，即

$$I=L _ { 2 } \circ \lambda \circ { L _ { 1 } } ^ { - 1 }$$

则

$$
L=L_2^{-1} \circ L_1=\lambda
$$

即证唯一性. $\square$

### 2. 几类重要的分式线性同构

**定义** 设区域$A,B\subseteq\overline{\mathbb{C}}$，且存在Mobius变换$f$使得$f(A)=B$，则称$A$和$B$是分式线性同构的，$f$称为这两个区域间的分式线性同构映射. 

**例（例4.4.4）** 上半平面到单位圆盘的分式线性同构

令

$$f(z)=\frac{z-i}{z+i}$$

则对任意$z\in \mathbb{R}$，$|f(z)|=1$，而Mobius变换将$\overline{\mathbb{C}}$中的圆映成圆，则$f$将实轴映成单位圆周. 

由基本的几何关系和复数相除的模长公式，对于任何上半平面的$z$，有$|f(z)|\leq 1$，即上半平面的像集在单位圆及其内部. 对于任何严格下半平面的$w$，有$|f(w)| > 1$，即下半平面的像在单位圆外部. 这就说明了上半平面的像就是单位圆，即$f$是上半平面到单位圆盘的分式线性同构. $\square$

**定理** $\overline{\mathbb{C}}$中任何两个圆盘分式线性同构. 

**证明** $\overline{\mathbb{C}}$中的圆盘要么是通常意义上的圆盘，要么是一条直线的单侧区域. 

由于直线单侧区域都可以通过旋转、平移得到上半平面，则所有直线单侧区域分式线性同构. 

任何通常意义上的圆盘都可以通过平移和以原点为中心的放缩得到单位元，故所有通常意义上的圆盘分式线性同构. 

由前面的例，单位圆盘和上半平面分式线性同构，则所有$\overline{\mathbb{C}}$中的圆盘分式线性同构. $\square$

**命题（例4.4.5）** 任何从上半平面到单位圆盘的分式线性同构都具有形式

$$f(z)=e^{i\theta}\frac{z-w}{z-\overline{w}}$$

其中$\Im (w) > 0$. 

**证明** 设$f$为从上半平面到单位圆盘的分式线性同构，令$w=f^{-1}(0)$，则由Mobius变换的保对称性，$f(\overline{w})=\infty$，则$f$具有形式

$$f(z)=A\frac{z-w}{z-\overline{w}}$$

由于Mobius变换和它的逆都是连续的，故其将区域的边界点映为像集的边界点，则由几何关系知$|A|=1$，即证原结论. $\square$

**命题** 单位圆盘的分式线性自同构都具有形式

$$f(z)=e^{i\theta}\frac{z-w}{1-\overline{w}z}.$$

**证明** 设$f$为单位圆盘的分式线性自同构，令$w$满足$f(w)=0$，则$w$关于单位圆的对称点$1/\overline{w}$满足$f(1/\overline{w})=\infty$，则$f$具有形式

$$f(z)=A\frac{z-w}{z-1/\overline{w}}=-A\overline{w}\frac{z-w}{1-\overline{w}z}$$

与前一命题证明过程同理，$f$将单位圆周映成单位圆周，则由$|f(1)|=1$代入得$|A\overline{w}|=1$，即得原式. $\square$

**命题** 任何上半平面的分式线性自同构都具有形式

$$f(z)=\frac{az+b}{cz+d},\ a,b,c,d\in \mathbb{R}.$$

**证明** 令

$$f_0(z)=\frac{z-i}{z+i}$$

则$f_0$将上半平面映为单位圆盘，设$f$为上半平面的分式线性自同构，则$f_0 \circ f\circ f_0^{-1}$为单位圆盘的分式线性自同构，记$\lambda=f_0 \circ f\circ f_0^{-1}$，则$f$具有形式

$$f=f_0^{-1}\circ\lambda\circ f_0$$

设$\lambda$为

$$\lambda(z)=e^{i\theta}\frac{z-re^{it}}{1-re^{-it}z}$$

取矩阵表示：

$$
F_0 = \begin{pmatrix}1 & -i\\1 & i\end{pmatrix},\quad 
F_0^{-1} = \begin{pmatrix}i & i\\-1 & 1\end{pmatrix},\quad
\Lambda = \begin{pmatrix}e^{i\theta} & -a e^{i\theta}\\-\bar{a} & 1\end{pmatrix},\; a=r e^{it}.
$$

计算复合变换的矩阵 $M = F_0^{-1}\Lambda F_0$：

$$
\begin{aligned}
M &= \begin{pmatrix}i & i\\-1 & 1\end{pmatrix}
\begin{pmatrix}e^{i\theta} & -a e^{i\theta}\\-\bar{a} & 1\end{pmatrix}
\begin{pmatrix}1 & -i\\1 & i\end{pmatrix} \\
&= \begin{pmatrix}
i\bigl(e^{i\theta}(1-a)+(1-\bar{a})\bigr) & e^{i\theta}(1+a)-(1+\bar{a}) \\
e^{i\theta}(1-a)-(1-\bar{a}) & -i\bigl(e^{i\theta}(1+a)+(1+\bar{a})\bigr)
\end{pmatrix}.
\end{aligned}
$$

令 $u = 1-a$, $v = 1+a$, 则 $\bar{u}=1-\bar{a}$, $\bar{v}=1+\bar{a}$，有

$$
M = \begin{pmatrix}
i(e^{i\theta}\bar{u}+u) & e^{i\theta}\bar{v}-v \\
e^{i\theta}\bar{u}-u & -i(e^{i\theta}\bar{v}+v)
\end{pmatrix}.
$$

乘以因子 $k = e^{-i\theta/2}$ 得

$$
kM = \begin{pmatrix}
i(e^{i\theta/2}\bar{u}+e^{-i\theta/2}u) & e^{i\theta/2}\bar{v}-e^{-i\theta/2}v \\
e^{i\theta/2}\bar{u}-e^{-i\theta/2}u & -i(e^{i\theta/2}\bar{v}+e^{-i\theta/2}v)
\end{pmatrix}.
$$

由于 $e^{i\theta/2}\bar{u} = \overline{e^{-i\theta/2}u},\; e^{i\theta/2}\bar{v} = \overline{e^{-i\theta/2}v}$，每个元素均为实数. 因此 $f$ 对应实矩阵，即存在 $a,b,c,d\in\mathbb{R}$ 使得 $f(z)=\dfrac{az+b}{cz+d}$. $\square$
