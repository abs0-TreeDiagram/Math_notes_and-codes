## 二、Mobius变换

Mobius变换也称分式线性函数，最早于出现在德国数学家A. Möbius的论文(1827年)中. 它是一类应用广泛的共形映射. 例如，莫比乌斯变换与狭义相对论中的洛伦兹变换等价，具体地，黎曼球面
可与时空中的光锥对应，而作用在光锥上的洛伦兹群正好与莫比乌斯变换群同构(这里不作进一步论述). 此联系展示了复函数理论与现代物理的惊人统一性. 

### 1. Mobius变换的定义

**定义（定义4.2.1）** 函数类

$$\left\{f(z)=\frac{az+b}{cz+d}:a,b,c,d\in \mathbb{C},ad\ne bc\right\}$$

中的元素称为Mobius变换. 

**注** 若上面的定义中允许$ad=bc$，此时$(az+b)/(cz+d)$是定义域为$\mathbb{C}\backslash \{-d/c\}$的常值函数. 定义排除了这种退化情况. 

当$c=0$时，$ad\ne 0$，此时变换为平移、旋转和放缩的复合，是全纯的线性函数. 

### 2. Mobius变换的几何意义

依次复合下列映射：

$$z\mapsto cz,\ z\mapsto z+d,\ z\mapsto \frac{1}{z},\ z\mapsto rz,\ z\mapsto z+s$$

所得映射为

$$f(z)=\frac{r+s(cz+d)}{cz+d}=\frac{scz+sd+r}{cz+d}$$

这是Mobius变换. 数乘、加法的几何意义显然是放缩旋转和平移, 而$1/z=\bar{z}/|z|^2$，其意义是先取共轭，然后将单位圆内外翻转. 

### 3. 扩充复平面上的Mobius变换

对于Mobius变换

$$f(z)=\frac{az+b}{cz+d}$$

显然$z\to -d/c$时$f(z)\to \infty$，故可以定义$f(-d/c)=\infty$，而$z\to \infty$时$f(z)\to a/c$，故可以定义$f(\infty)=a/c$，这样就把Mobius变换的定义延拓到了$\overline{\mathbb{C}}$上. 

**命题** Mobius变换是$\overline{\mathbb{C}}$的同胚映射，即它是双射且有连续的逆. 

**证明** $c=0$时显然，$c\ne 0$时，双射性可直接验证，且易得其逆映射为

$$w\mapsto \frac{dw-b}{a-cw}$$

则它是$\mathbb{\overline{C}}$上的同胚映射. $\square$

### 4. Mobius变换的群结构

**命题** Mobius变换全体在复合运算下构成一个群. 

**证明** 封闭性：设

$$f(z)=\frac{az+b}{cz+d},\ g(z)=\frac{mz+n}{pz+q}$$

则

$$\begin{aligned}
    f\circ g(z)&=\frac{a\frac{mz+n}{pz+q}+b}{c\frac{mz+n}{pz+q}+d}\\
    &=\frac{amz+an+bpz+bq}{cmz+cn+dpz+dq}\\
    &=\frac{(am+bp)z+(an+bq)}{(cm+dp)z+(cn+dq)}
\end{aligned}$$

而

$$(am+bp)(cn+dq)=amcn+amdq+bpcn+bpdq$$

$$(an+bq)(cm+dp)=ancm+andp+bqcm+bqdp$$

两者相减得到

$$ad(mq-np)+bc(pn-qm)=(mq-np)(ad-bc)$$

由Mobius变换的定义上面$RHS$两个因式都不为零，故$f\circ g$为Mobius变换. 

结合律：任何映射的复合都满足结合律. 

幺元：显然包含. 

逆元：前文已述. 

故Mobius变换全体构成群. $\square$

**注** 该群不是交换群. 考虑$L : z \mapsto z + 1, \, \, \, M : z \mapsto { \frac { 1 } { z } }$，显然二者不交换. 

****

**命题** Mobius变换全体构成的群同构于

$$\mathrm { P G L } _ { 2 } \left( \mathbb { C } \right) : = \mathrm { G L } _ { 2 } \left( \mathbb { C } \right) \Big / \left\{ \left( \begin{matrix} { a } & { 0 } \\ { 0 } & { a } \\ \end{matrix} \right), \right. \left. a \in \mathbb { C } \right\}.$$

$\mathrm { G L } _ { 2 } \left( \mathbb { C } \right) $表示行列式非零的二阶复矩阵全体构成的集合. 

**证明** 设Mobius变换全体构成的集合为$M$，记$Q=\{aI:a\in \mathbb{C}\}$令映射

$$\begin{aligned}
    \phi:M&\to \mathrm { G L } _ { 2 } \left( \mathbb { C } \right) /Q\\
    \left(z\mapsto\frac{az+b}{cz+d}\right)&\mapsto  \left(\begin{matrix}
        a& b\\c&d
    \end{matrix}\right)Q
\end{aligned}$$

设

$$f(z)=\frac{az+b}{cz+d},\ g(z)=\frac{mz+n}{pz+q}$$

则前面已证

$$\begin{aligned}
    f\circ g(z)&=\frac{(am+bp)z+(an+bq)}{(cm+dp)z+(cn+dq)}
\end{aligned}$$

则

$$\phi(f\circ g)=\left(\begin{matrix}
        am+bp& an+bq\\cm+dp&cn+dq
    \end{matrix}\right)Q=\left(\begin{matrix}
        a& b\\c& d
    \end{matrix}\right)\left(\begin{matrix}
        m  & n \\ p&q
    \end{matrix}\right)Q=\phi(f)\phi(g)$$

这说明$\phi$是群同态. 

若$\phi(f)=\phi(g)$，则

$$\left(\begin{matrix}
        a& b\\c& d
    \end{matrix}\right)Q=\left(\begin{matrix}
        m  & n \\ p&q
    \end{matrix}\right)Q$$

即存在$w\in \mathbb{C}$使得

$$\left(\begin{matrix}
        a& b\\c& d
    \end{matrix}\right)=w\left(\begin{matrix}
        m  & n \\ p&q
    \end{matrix}\right)$$

这说明

$$\frac{az+b}{cz+d}=\frac{wmz+wn}{wpz+wq}=\frac{mz+n}{pz+q}$$

即$f=g$. 则$\phi$是单射. 满射性显然，则$\phi$是群同构. $\square$

## 三、保角性、保圆性和保对称性

### 1. 保角性

**命题** Mobius变换在$\mathbb{C}$中任何点都保角. 

**证明** 设$f(z)=(az+b)/(cz+d)$，则对于任意$z\in \mathbb{C}$，$z\ne -d/c$，有

$$f'(z)=\frac{ad-bc}{(cz+d)^2}\ne 0$$

故共形，进而保角. 

要证$f$在$z=-d/c$处保角，只需证$1/f$在$z=-d/c$处保角. 

$$\frac{1}{f}=\frac{cz+d}{az+b}$$

这也是Mobius变换，则由前面推导其在$z=-d/c$处保角. 

要证$f$在$\infty$处保角，只需证$f(1/z)$在$z=0$处保角. 

$$f(\frac{1}{z})=\frac{a(1/z)+b}{c(1/z)+d}=\frac{a+bz}{c+dz}$$

显然其在$z=0$处保角. 

综上$f$在$\overline{}{\mathbb{C}}$上保角. $\square$

### 2. 保圆性

**定义（定义4.3.3）** 通常意义上的$\mathbb{C}$中的直线和圆称为$\overline{\mathbb{C}}$上的直线. 

**定理（定理4.3.4）** Mobius变换将$\overline{\mathbb{C}}$中的圆映为$\overline{\mathbb{C}}$中的圆. 

**证明** 前文已经验证Mobius变换可表示为平移、旋转、放缩和反演（$z\mapsto 1/z$）的复合，前三个显然有保圆性，只需证反演变换也有保圆性. 

$\overline{\mathbb{C}}$中的圆方程可以表示为

$$A \! \left( x ^ { 2 } + y ^ { 2 } \right) + B \, x + C \, y + D = 0 \,,$$

在复平面上$z=x+iy$，上式可表为

$$Az\overline{z}+\frac{B}{2}(z+\overline{z})+\frac{C}{2i}(z-\overline{z})+D=0$$

$$Az\overline{z}+\frac{B-Ci}{2}z+\frac{B+Ci}{2}\overline{z}+D=0$$

则它在$z\mapsto1/z$下的像为

$$A\cdot\frac{1}{z\overline{z}}+\frac{B-Ci}{2}\cdot\frac{1}{z}+\frac{B+Ci}{2}\cdot\frac{1}{\overline{z}}+D=0$$

$$A+\frac{B-Ci}{2}\overline{z}+\frac{B+Ci}{2}z+Dz\overline{z}=0$$

这也是$\overline{\mathbb{C}}$上的圆. $\square$

事实上，反演变换在Riemann球面的视角看，就是把球面沿实轴旋转了$\pi$，则显然具有保圆性. 

**问题** 记$E={B-Ci}/{2}$，$A\ne 0$时，方程

$$A \, z \, \overline { { z } } + E \, z + \overline { { E } } \, \overline { { z } } + D = 0$$

确定的圆的圆心和半径为何？

**解** 以$c$为圆心、$r$为半径的圆可表示为

$$(z-c)(\overline{z}-\overline{c})=r^2$$

展开得

$$z\overline{z}-\overline{c}z-c\overline{z}+c\overline{c}=r^2$$

整理题中原式

$$ \, z \, \overline { { z } } + \frac{E}{A} \, z + \frac{\overline { { E } }}{A} \, \overline { { z } } + D = 0$$

则

$$c=-\frac{\overline{E}}{A},\ r^2=\frac{E\overline{E}}{A^2}-D.\ \square$$

### 3. 保对称性

**定义（定义4.3.5）** 设$\Gamma=\partial B(z_0,r)$为$\mathbb{C}$中的圆（不包括直线），若$z_0,a,b$共线、$a,b$在$z_0$的同侧且

$$|a-z_0||b-z_0|=r^2$$

则称$a,b$关于圆$\Gamma$对称. 

**引理** （圆幂定理）设平面上$\odot O$外一点$P$引出的两条射线分别交$\odot O$于$A,B$和$C,D$，则

$$PA\cdot PB=PC\cdot PD. $$

**证明** 由四点共圆对角互补的性质和相似三角形的相关结论即证. $\square$

**命题** $a,b$关于圆$\Gamma=\partial B(z_0,r)$对称等价于$\overline{\mathbb{C}}$中过$a,b$的所有圆与$\Gamma$正交，即相交处道路的夹角为直角. 

**证明** $\implies$：过$a,b$的直线显然满足要求，只需考虑圆. 

设$\odot P$是某个过$a,b$的圆，由圆幂定理，从$z_0$引$\odot P$的两条切线，（由切线长定理它们长度相等），设切线长为$l$，则

$$l^2=|a-z_0||b-z_0|=r^2$$

故切点在$\Gamma$上，即证两圆正交. 

$\impliedby$：对于直线，过$a,b$的直线与$\Gamma$正交则必过$z_0$，故$a,b,z_0$共线. 

对于圆，过$a,b$的圆与$\Gamma$正交，说明从$z_0$引$\Gamma$的切线切点在$\Gamma$上，由圆幂定理$|a-z_0||b-z_0|=r^2$. $\square$

**定理** 设$f$为Mobius变换，点$a,b$关于$\overline{\mathbb{C}}$上的圆$\Gamma$对称，（由保圆性，$f(\Gamma)$还是$\overline{\mathbb{C}}$上的圆，）则$f(a),f(b)$关于圆$f(\Gamma)$对称. 

**证明** 对于任何过$f(a),f(b)$的圆$B$，由于Mobius变换的逆还是Mobius变换，则存在过$a,b$的圆$B_0$使得$f(B_0)=B$. 

由于$B$与$\Gamma$正交，而Mobius变换保角，则$f(B)$与$f(\Gamma)$正交. 由上一个命题即得结论. $\square$
