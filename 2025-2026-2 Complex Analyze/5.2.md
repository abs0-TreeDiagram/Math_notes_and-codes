## 二、Rokovsky函数

### 1. Rokovsky函数的定义和基本性质

**定义** 函数

$$
w(z)=\frac{1}{2}(z+\frac{1}{z})
$$

称为**Rokovsky函数**. 

**练习（1）** 说明Rokovsky函数在$\mathbb { C } \backslash \{ 0, \pm 1 \}$中的任何点上共形. 

**证明** 

$$
w'(z)=\frac{1}{2}(1-\frac{1}{z^2})
$$

对于任何$z\in  \mathbb { C } \backslash \{ 0, \pm 1 \}$，显然有$w'(z)\ne 0$，故$w(z)$在$z$处共形. $\square$

**（2）** 若令$w(0)=\infty$，Rokovsky函数$w(z)$在$z=0$处是否具有保角性？

**解** 在$z=0$附近考虑

$$
\frac{1}{w(z)}=\frac{2}{z+1/z}=\frac{2z}{z^2+1}
$$

显然$1/w$在$z=0$处全纯，且容易求得其在$z=0$处导数为$2\ne 0$，故它是保角的. 进而$w(z)$在$z=0$也是保角的. 

**（3）** 若令$w(\infty)=\infty$，它在$\infty$是否具有保角性？

**解** 考虑函数

$$
\frac{1}{w(1/z)}=\frac{2}{1/z+z}=\frac{2z}{1+z^2}
$$

与$(2)$相同，该函数在$z=0$处全纯且导数不为零，故在$z=0$处保角，进而$w(z)$在$\infty$处保角. $\square$

**命题5.2.1** 设区域$D\subseteq \mathbb{C}\backslash \{0\}$，则Rokovsky函数$w(z)$在$D$上单叶等价于$D$中不存在$z_1,z_2$使得$z_1z_2=0$. 

**证明** $w$在$D$上非单叶

$\iff$ 存在$z_1,z_2\in D$使得$w(z_1)=w(z_2)$. 

$\iff$ 存在$z_1,z_2\in D$使得

$$
\begin{aligned}
    &z_1+\frac{1}{z_1}=z_2+\frac{1}{z_2}\\
    \iff & z_1-z_2=\frac{z_1-z_2}{z_1z_2}\\
    \iff&z_1z_2=1
\end{aligned}
$$

即证原命题两方面的逆否命题，即证原命题. $\square$

**推论** Rokovsky函数在单位圆盘外部区域是单叶的. 

### 2.Rokovsky函数的几何直观

记$z=r\exp(i\phi)$，则

$$
\begin{aligned}
    w(z)&=\frac{1}{2}(z+\frac{1}{z})\\
    &=\frac{1}{2}(re^{i\phi}+\frac{e^{-i\phi}}{r})\\
    &=\frac{1}{2}(r\cos \phi+ir\sin \phi+\frac{1}{r}\cos \phi-\frac{i}{r}\sin \phi)\\
    &=\frac{1}{2}(r+\frac{1}{r})\cos \phi+i\frac{1}{2}(r-\frac{1}{r})\sin \phi
\end{aligned}
$$

**练习** 根据上式说明

**（1）** 对于$r_0 > 1$，$w$将$\partial B(0,r_0)$映成半轴为

$$
a _ { r _ { 0 } } = { \frac { 1 } { 2 } } \left( r _ { 0 } + { \frac { 1 } { r _ { 0 } } } \right),\ b _ { r _ { 0 } } = \frac { 1 } { 2 } \left( r _ { 0 } - \frac { 1 } { r _ { 0 } } \right)
$$

焦点为$\pm 1$，离心率为$2/(r_0+1/r_0)$的椭圆. 

**证明** $r_0 > 1$时$w(r_0,\phi)$的实部和虚部都大于零，对于$\phi\in [0,2\pi]$，其表达式就是上述椭圆的极坐标表达式. 由圆锥曲线的基本知识即得焦点和离心率. $\square$

**（2）** $w$将单位圆周映成线段$[-1,1]$. 

**证明** 由$r=1$得$w(r,\phi)$的虚部为零，实部为$\cos \phi$，由$\phi$的取值范围即得结论. $\square$

**（3）** $w$将射线$\{\phi=\phi_0\}$映成$\mathbb{C}$中双曲线

$$
\frac { x ^ { 2 } } { \operatorname { c o s } ^ { 2 } \phi _ { 0 } } \, - \, \frac { y ^ { 2 } } { \operatorname { s i n } ^ { 2 } \phi _ { 0 } } \, = \, 1
$$

的一支. 其中$z=x+iy$. 

**证明** 直接将$w(r,\theta_0)$的实部、虚部代入$x,y$验证即得满足双曲线方程. 又由于$w$是连续的，故将连续的射线映成连续的道路，进而为双曲线中的一支. $\square$

Rokovsky函数的另外一个有意思的单叶区域由通过 $z = \pm 1$ 两点的(内)圆及与之相切于 $z = 1$ 点的(外)圆围成(如下图)。它的像呈“机翼”状。注意到，在点 $z = 1$ 处(非共形点)外圆的像产生了尖点，而内圆被映成了连接这两点的曲线段。改变这两个圆圆心的位置可以得到不同的“机翼”形状。

![](https://files.mdnice.com/user/99943/ce632656-38df-4eca-83f4-3f8593f4d07a.png)

俄国数学家Rokovsky借助这一特殊图像，成功构建了飞行器空气动力学的首个可计算的数学模型理论。这也是此函数被称为Rokovsky函数的原因.
