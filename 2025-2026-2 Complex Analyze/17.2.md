## 二、全纯延拓的定义

### 1. 典范元

**定义17.2.1（典范元）** （1）设$U=B(a,r)$为开圆盘，$f$为$U$上的全纯函数，且不存在$r' > r$和函数$g$，使得$g$在$V:=B(a,r')$上全纯且$g|_{B(a,r)}=f$，则称偶对$F:=(U,f)$为**典范元**. 

（2）$a$称为典范元的**中心**，$U$称为典范元的**圆盘**，$r$称为典范元的**半径**. 对于$z\in U$，$f(z)$称为典范元$F$在$z$处的**值**. 

（3）两个典范元相等当且仅当它们的圆盘相等且在圆盘上任何点的值也相等（即典范元中的函数相等）. 

由复Taylor级数的基本性质可以证明，对于典范元$F=(U,f)$，若其半径为$r$，中心为$a$，那么$f$在$a$点的Taylor级数的收敛半径就是$r$. 

**定义（以$\infty$为中心的典范元）** 设$U$为$\infty$的$r$-邻域，$f$在$U$上全纯（进而在$\infty$全纯，进而其在原点处洛朗级数的最高非零次项为常数项），且不存在$\infty$的更大邻域使得$f$能够继续全纯延拓，则称$F=(U,f)$为一个（以$\infty$为中心的）典范元. 

**练习** 说明$( U : = \{ | z - i | < 1 \}, \, \cot z )$和$( V : = \{ | z - i - \pi | < 1 \}, \, \cot z )$都是典范元且不相等. 

**证明** $\cot z$的所有奇点为 $k\pi, k\in \mathbb{Z}$. 故不存在以$i$为圆心且半径大于$1$的开圆盘使得$\cot z$在其上全纯，故$(U,\cot z)$为典范元（否则将包含奇点$0$）. 同理可证$(V,\cot z)$也是典范元（更大的圆盘将包含奇点$\pi$）. 这两个典范元的圆盘不同，故不相等. $\square$

### 2. 全纯延拓

**定义17.2.2（直接全纯延拓）** 若典范元$F_1=(U,f),F_2=(V,g)$的圆盘有交，且在$U\cap V$上有$f=g$，则称$F_1,F_2$互为**直接全纯延拓**. 

**练习** 给定典范元$F=(U,f)$及开圆盘$V$，若$U\cap V$非空，则最多存在一个典范元$G=(V,g)$使得$G,F$互为直接全纯延拓. 

**证明** 假设$G_1=(V,g_1)$也与$F$互为直接全纯延拓，则在$U\cap V$上有$g=f=g_1$，由全纯函数的唯一性定理，在$V$上有$g=g_1$，这说明$G=G_1$，即证唯一性. $\square$

**定义17.2.3（全纯延拓）** 对于典范元$F=(U,f)$与$G=(V,g)$，若存在有限个典范元$F_k=(U_k,f_k)$，$k=1,2,...,n$，$F_1$与$F$、$F_n$与$G$互为直接全纯延拓，且对于$k=2,3,...,n$，$F_{k-1}$与$F_k$互为直接全纯延拓，则称$F$与$G$互为**全纯延拓**.

**例** 分析奇点分布并由基本几何关系可得，典范元$( U : = \{ | z - i | < 1 \}, \, \cot z )$和$( V : = \{ | z - i - \pi | < 1 \}, \, \cot z )$不互为直接全纯延拓，但互为全纯延拓. 如图：

**问题** 全纯延拓的定义（17.2.3）中，若典范元$U,V$的圆盘有交，是否要求在交集上$f=g$？

**答** 不要求. 实际上也不一定相等. 

**例** 设$U_0=B(1,1)$，$U_1=B(i,1)$，$U_2=B(-1,1)$，$U_{-1}=B(-i,1)$，$f_0,f_1,f_{-1}$分别为$U_0,U_1,U_{-1}$上由对数主支确定的平方根函数的全纯分支：

$$
f_0 \left( z \right) = \sqrt { r } \ e ^ { i \, \phi / 2 } \,, \ \ z = r \, e ^ { i \, \phi } \,, \ \ - \pi / 2 < \phi < \pi / 2
$$

$$
f _ { 1 } ( z ) = \sqrt { r } \, \, \, e ^ { i \, \phi / 2 } \,, \, \, \, \, \, z = r \, \, e ^ { i \, \phi } \,, \, \, \, \, \, 0 < \phi < \pi

$$

$f_2$为平方根函数在$U_2$上的全纯分支：

$$
f _ { 3 } ( z ) = \sqrt { r } ~ e ^ { i \, \phi / 2 } \,, ~ ~ z = r \, e ^ { i \, \phi } \,, ~ ~ \pi / 2 < \phi < 3 \, \pi / 2
$$

$f_{-1}$为平方根函数在$U_4$上的全纯分支：

$$
f _ { - 1 } ( z ) = \sqrt { r } \, \, \, e ^ { i \, \phi / 2 } \,, \, \, \, \, \, z = r \, e ^ { i \, \phi } \,, \, \, \, \, \, - \pi < \phi < 0
$$

$f_{-2}$为平方根函数在$U_2$上的全纯分支：

$$
f _ { - 2 } ( z ) = \sqrt { r } \ e ^ { i \, \phi / 2 } \,, \ \ z = r \, e ^ { i \, \phi } \,, \ \ - 3 \, \pi / 2 < \phi < - \pi / 2 \,.
$$

显然，典范元$(F_2,f_2)$和$(F_2,f_{-2})$不等，但它们互为全纯延拓.
