## 一、辐角函数

### 1. 辐角函数的定义

熟知复数的辐角有无穷多个取值. 事实上对于$z\ne 0$，满足

$$
\begin{aligned}
    |z|\cos \theta = \Re (z)\\
    |z|\sin \theta = \Im (z)
\end{aligned}
$$

的$\theta$都是$z$的辐角. 

为了保证确定性，我们需要在特定区域内将幅角函数$\mathrm{Arg}\ z$ “分解”为若干个单值的、连续的函数. 

**定义6.1.1**（幅角函数的连续分支）称区域$D \subset \mathbb { C } \backslash \{ 0 \}$上的单值连续函数 $f (z)$为幅角函数$\mathrm{Arg}\ z$的一个连续分支，如果 $∀ z ∈ D$都存在某$k ∈ \mathbb{Z}$（可能依赖于$z$）使得$f (z) = \arg z + 2 k \pi$. 在语意清晰的情况下，也可记作$\arg z$.

**问题** 辐角函数的连续分支可能全纯吗？

**答** 由于区域上的全纯函数一定是开映射，而辐角函数的像集是$\mathbb{R}$的子集，没有内点，故辐角函数的连续分支一定不全纯. 

### 2. 辐角函数的割线

对于$z\ne 0$，考虑围绕$z$且不包含$0$的Jordan闭道路，选定闭道路上动点$w$，则$w$绕闭道路运动一周并使其辐角在此过程中连续变化，则运动前与运动后$w$的辐角值相等. 若$z=0$或$z=\infty$，则运动一周后辐角值将相差$2\pi$. 

因此对于辐角函数，$0,\infty$是两个特殊点. 若区域$D$上能够定义$\mathrm{Arg}$的单值连续分支，则$D$中不能包含围绕$0$或$\infty$的闭曲线，即：$D$中不能包含$0$和$\infty$的任何邻域. 

![](https://files.mdnice.com/user/99943/4eb1c61e-3b21-4d58-b348-d61d868b2010.png)

**定义** $\mathbb{C}$中任何连接$0$与$\infty$的无界Jordan曲线称为**辐角函数的割线**. 

**命题6.1.2** （1）设$K$是辐角函数的割线，给定$z_1\in \mathbb{C}\backslash K$与$k\in \mathbb{Z}$，则存在$\mathbb{C}\backslash K$上的辐角函数的唯一连续分支$f(z)$使得$f(z_1)=\arg z_1+2k\pi$. 

（2）对于任意$b\in K\backslash \{0\}$，$K$将$b$的任何足够小的邻域$B$分成两个单连通分支$B_+$和$B_-$，

$$
|\lim_{z\to b,z\in B_+}f(z)-\lim_{z\to b,z\in B_-}f(z)|=2\pi. 
$$

## 二、对数函数

### 1. 对数函数的定义

**定义6.2.1：复对数** 对任何$z\in \mathbb{C}$，若$w\in C$满足$z=e^w$，则称$w$为$z$的对数，记作$w\in \mathrm{Ln}\ z$. 

在上面的定义中，设$w=u+vi$，$z=re^{i\theta}$，则

$$
re^{i\theta}=\exp(u+iv)=\exp(u)\cdot \exp(iv)
$$

则

$$
\begin{aligned}
    r=\exp(u)\iff u=\ln r\\
    e^{i\theta}=e^{iv}\iff v=\theta+2k\pi,\ k\in \mathbb{Z}
\end{aligned}
$$

由于辐角函数是无穷多值的，故对数函数也是无穷多值的. 我们将函数

$$
z\mapsto \ln |z|+i\arg z
$$

称为$\mathrm{Ln}\ z$的主值，记作$\ln$. 其中$\arg\in (-\pi,\pi]$为辐角主值. 则

$$
\mathrm{Ln}\ z=\ln z+i2k\pi. 
$$

**注** 对数函数的虚部是幅角函数，所以上图也可作幅角函数多值性演示。

### 2. 对数函数的性质

**命题6.2.2** （1）对任意$z\in \mathbb{C}\backslash \{0\}$，有$\exp(\mathrm{Ln}\ z)=z$. 

**证明** 

$$
\exp(\mathrm{Ln}\ z)=\exp(\ln |z|+i\arg z+i2k\pi)=|z|e^{i\arg z}=z.\ \ \square
$$

（2）对任意$z_1,z_2\in \mathbb{C}\backslash\{0\}$，有

$$
\mathrm{Ln}\ (z_1z_2)= \mathrm { L n } \, z _ { 1 } + \mathrm { L n } \, z _ { 2 }
$$

$$
\mathrm { L n } \left( z _ { 1 } / z _ { 2 } \right) = \mathrm { L n } \, z _ { 1 } - \mathrm { L n } \, z _ { 2 }
$$

**证明** 

$$
\begin{aligned}
    \mathrm{Ln}\ (z_1z_2)&=\ln |z_1z_2|+i\arg (z_1z_2)+i2k\pi\\
    &=\ln |z_1|+\ln |z_2|+i\arg z_1+i\arg z_2+i2k\pi\\
    &= \mathrm { L n } \, z _ { 1 } + \mathrm { L n } \, z _ { 2 }
\end{aligned}
$$

$$
\begin{aligned}
    \mathrm { L n } \left( z _ { 1 } / z _ { 2 } \right)&=\ln |z_1/z_2|+i\arg (z_1/z_2)+i2k\pi\\
    &=\ln |z_1|-\ln |z_2|+i\arg z_1-i\arg z_2+i2k\pi\\
    &=\mathrm { L n } \, z _ { 1 } - \mathrm { L n } \, z _ { 2 }&&\square
\end{aligned}
$$

### 3. 对数函数的单值连续分支

**定义6.2.3** 设$f$在区域$D$上连续，且$\exp(f(z))=z$，则称$f$是对数函数的**单值连续分支**. 

与辐角函数相似，有

**命题6.2.4** 在复平面上取 $K$ 为辐角函数的割线，取定任意 $z_0 \in D = \mathbb{C} \setminus K$ 和 $k \in \mathbb{Z}$，则存在区域 $D$ 上对数函数 $\operatorname{Ln} z$ 的唯一单值连续分支 $f(z)$ 使得 $f(z_0) = \ln |z_0| + i(\arg z_0 + 2k\pi)$；对任意 $b \in K \setminus \{0\}$，割线 $K$ 将 $b$ 的任意充分小邻域 $B$ 分成两个单连通部分 $B_+$ 和 $B_-$，并且极限 $b_+ = \lim_{z \to b, z \in B_+} f(z)$ 和极限 $b_- = \lim_{z \to b, z \in B_-} f(z)$ 存在且相差$2\pi i$. 

**命题6.2.5** 设某个区域$D\subseteq\mathbb{C}\backslash \{0\}$上有单值对数全纯分支$\ln z$，则$\ln z$在$D$上全纯，且$(\ln z)'=1/z$. 

**证明** 任取$z_0\in D$，令$w_0=\ln z_0$，则$e^{w_0}=z_0$，记$f:z\mapsto e^z$为指数函数，由于$0\ne D$，则$D$包含在$f(\mathbb{C})$中. 

熟知$f$是$\mathbb{C}$上的全纯函数，进而是连续函数. 开集在连续函数下的原像还是开集，故$f^{-1}(D)$为开集（注意$f^{-1}$此处仅表示原像）. 显然对于$D$中每一点$z$，$\ln z$是$z$在$f$下的一个原像，则$\ln z$是$f(z)$的一个连续的逆映射. 对于任何$z$，$e^z=(e^z)'\ne 0$. 由定理4.1.6，

$$
(\ln f(z))'=\frac{1}{f'(z)}
$$

则$(\ln z)'=1/z$. 且$\ln$在$D$中每一点都可导，进而在$D$上全纯. $\square$

在提及某全纯函数时，如无特别说明，一般指的都是单值全纯函数或
者多值函数的某全纯分支. 

**练习** 寻找$\mathbb{C}\backslash \{x\leq 0\}$上的对数全纯分支，说明它可以把这个区域映成带状区域

$$
\{ ( 2 \, k - 1 ) \, \pi < \textrm { I m } ~ z < ( 2 \, k + 1 ) \pi \}. 
$$

**解** 考虑函数

$$
\begin{aligned}
    f_k:\mathbb{C}\backslash \{x\leq 0\}&\to \mathbb{C}\\
    z&\mapsto \ln |z|+i\arg z+i2k\pi
\end{aligned}
$$

它们是$\mathbb{C}\backslash \{x\leq 0\}$上的所有对数全纯分支. 

$\mathbb{C}\backslash \{x\leq 0\}$中的任何点可以表示为

$$
re^{i\theta},\quad r\in (0,\infty),\theta\in (-\pi,\pi). 
$$

则

$$
f_k(\mathbb{C}\backslash \{x\leq 0\})=\{r+i\theta+2ki\pi:r\in (0,\infty),\theta\in (-\pi,\pi)\}
$$

右边正是题目所说的带状区域. $\square$

**关于负数对数值的一段简史**

1712年至1713年间，约翰·伯努利(Johann Bernoulli)与莱布尼茨(G.W . Leibniz，此时莱布尼茨已达67岁高龄)之间就负数的对数值应该是什么发生了激烈的争论(以书信的形式)。伯努利断言它们是实数，并且$\ln-x = \ln x$; 而莱布尼茨则认为它们是虚数，并且$\ln-x ≠ \ln x$，特别地，$\ln-1 ≠ 0$。

三十多年后，1749年，欧拉(L. Euler)加入了这场争论，他在一篇论文中指出原争论双方的结论都是不对的，并给出了一个正确的解答：负数的对数是个无穷多值的集合。

作为对欧拉论文的回应，达朗贝尔(Jean le Rond d ' Alembert)在1752年也撰文讨论负数的对数值(正式发表于1761年)，他明确支持伯努利，反对莱布尼茨和欧拉。

彼时，达朗贝尔和欧拉有多项涉及数学发现的优先权等等的争端，这是其中一环。

**定义6.2.7：分支点** 给定复平面上的一个多值函数 $F(z)$，称 $z_0 \in \overline{\mathbb{C}}$ 为该函数的分支点，若存在 $\varepsilon > 0$，使得对于任意 $r\in  (0,\varepsilon)$，在 $z_0$ 邻域 $B(z_0, r)$ 内都存在一条内部包含 $z_0$ 的若当闭曲线 $C$ 及其上一点 $z_1$ 的函数值，函数 $F(z)$ 从点 $z_1$ 出发沿着曲线 $C$ 连续变化一圈回到点 $z_1$ 得到的函数值与给定值不同；如果存在最小的整数 $n > 1$，使得其沿 $B(z_0, \epsilon)$ 内任意内部包含 $z_0$ 的若当闭曲线连续变化 $n$ 圈得到的函数值与初始值相同，则称它为 $n-1$ 阶分支点；否则称为无穷阶分支点. 

显然，$\infty$和$0$都是对数函数的无穷阶分支点.

- 辐角函数与对数函数的定义
- 辐角函数与对数函数的割线：辐角函数的割线连接$0$与$\infty$，在去除割线的单连通区域上可以定义连续的辐角分支和全纯的对数分支. 
- 复对数满足与实对数相同的运算和求导性质.
